# 北美门前场景探索器（North America Door Lock Scenario Explorer）

面向北美住宅智能门锁的 AI 功能规划场景枚举工具。

将「人物 × 用房类型 × 时刻 × 住宅类型 × 事件 × 活动」6 个维度进行组合枚举，
每个场景可展开查看：按时间节点（回家前/开门/门内/出门前…）的详细用户旅程、
人物画像、关键痛点与 Moment of Truth。

## 在线访问

启用 GitHub Pages 后访问：

`https://derreckcoco.github.io/na-doorlock-scenario-explorer/`

## 本地运行

```bash
npx serve .
```

然后浏览器打开 http://localhost:3000（或终端提示的端口）。

> 不能直接双击 `index.html` 打开，需要通过本地/远程 HTTP 服务器（因为使用了 ES Module）。

## 维度说明

| 维度 | 说明 |
|------|------|
| 人物 | 常住人 / 访客 / 服务人员 / 物流人员，共 14 种角色 |
| 用房类型 | 自住 / 租赁 / Airbnb / 空置 / 商用 |
| 时刻 | 回家前 / 开门 / 门内 / 出门前 / 离家 / 长期离家 / 居家 / 夜间 |
| 住宅类型 | 独栋 / 联排 / 公寓 / 豪宅 / ADU / 度假屋 / 55+ / Condo |
| 事件 | 通勤 / 上学 / 遛狗 / 育儿 / 取包裹 等 14 种 |
| 活动 | 派对 / 节日 / 装修 / 搬家 / 无特定活动 |

## 使用建议

1. 用左侧筛选器缩小到具体人物/时刻组合
2. 点击卡片展开，查看该场景的完整旅程与痛点
3. 关联度分数（高/中/低）用于判断该组合在现实中的常见程度，仅供参考排序，不代表功能优先级

---

本工具仅包含通用场景枚举框架，不含具体产品定价/竞品/业务策略分析。
